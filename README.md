# Pharmacogenomics Variant Analysis

## Project Overview
This project investigates clinically relevant pharmacogenomic variants associated with drug metabolism, efficacy, and toxicity. The analysis focuses on genes involved in drug response and aims to demonstrate bioinformatics workflows for pharmacogenomic research.

## Objectives
- Identify pharmacogenomic variants in key genes.
- Explore the relationship between genetic variants and drug response.
- Develop reproducible bioinformatics workflows.
- Build a portfolio-ready pharmacogenomics project.

## Genes of Interest
- CYP2D6
- CYP2C19
- CYP3A4
- TPMT
- DPYD
- SLCO1B1

## Tools and Technologies
- Python
- Pandas
- Jupyter Notebook
- Git and GitHub
- Linux/Unix
- Matplotlib and Seaborn

## Expected Outputs
- Variant summary tables
- Data visualizations
- Drug-gene interaction analyses
- Reproducible analysis notebooks
